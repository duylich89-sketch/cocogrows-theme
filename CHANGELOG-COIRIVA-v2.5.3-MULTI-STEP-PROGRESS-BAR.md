# Coiriva v2.5.3 – CRO Ultimate | Multi-Step Progress Bar

## Summary
Added a lightweight, app-free multi-step cart rewards progress bar to the product buy box.

## Added
- `snippets/coiriva-progress-bar.liquid`
- `assets/coiriva-progress-engine.css`
- `assets/coiriva-progress-engine.js`

## Modified
- `snippets/coiriva-buybox-pro-addons.liquid`
- `layout/theme.liquid`
- `config/settings_schema.json`

## Features
- 3-step rewards system:
  - Free Shipping goal
  - Best Value goal
  - Premium Bonus goal
- Dynamic cart subtotal detection via `/cart.js`
- Smooth progress fill animation
- Goal markers with unlocked states
- Responsive mobile layout
- Theme Editor settings for thresholds
- No paid app required
- No external library
